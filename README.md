# Android Social Media App

A social media platform Android app developed using **Object-Oriented Programming (OOP)** principles. Users can create accounts, post updates, like and comment on posts, and interact with other users.

---

## Features
- User registration and login
- Create, edit, and delete posts
- Like and comment on posts
- User profiles with basic info
- Object-Oriented Programming structure: Classes, Inheritance, Encapsulation, Polymorphism
- Modular and reusable code

---

## Tech Stack
- **Language:** Java 
- **IDE:** Android Studio
- **Database:** Firebase (depending on your implementation)
- **Architecture:** OOP
